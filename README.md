# Simple Image Search Engine

An implementation of a simple image search engine in Python.


## Requirements

- Python 3.6
- Required packages are listed in `requirements.txt`

## Installation

Install the required packages:
```
pip install -r requirements.txt
```

Download sample images from [here](http://bitly.com/holiday-small).
Extract the zip file and put the `images` folder in `static` folder.

## Usage

```
python image_search.py
```

Dataset is taken from: [INRIA Holiday Dataset](http://lear.inrialpes.fr/people/jegou/data.php)
